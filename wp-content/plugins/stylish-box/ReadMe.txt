Change the post, page (the content) CSS into the Colorful Box CSS Shadow

== Description ==

Choose from 8 stitched bordered boxes featuring different types of shadows each supporting full responsiveness, styled in pure CSS individually.

== Installation ==

1. Upload the files to the `/wp-content/plugins/stylish-box/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Settings->Stylish Box screen to configure the plugin
