OWL Carousel Slider is a Responsive Carousel Slider Plugin With Touch Swipe Navigation. Incredibly Easy to Install and Use.

== Description ==

OWL Carousel Slider is a highly customizable and full responsive  Carousel Slider plugin for WordPress. This Carousel Slider plugin can be displayed  a slider anywhere in your wordpress site. 

You can able to use 40+ parameters for customizations with shortcode, which makes it extremely configurable and user friendly. You can use as many instance of this OWL Carousel slider in your site without any type of conflict.

OWL Carousel slider also supports YouTube or Vimeo videos that helps to display video galleries.


**New Features of the plugin includes**

*   Control Displayable Slides Item According Device Width( Viewport ). ( NEW )
*   Multiple Instances on One Page. ( NEW )
*   Touch swipe navigation. ( NEW )
*   Infinity Slides loop. ( NEW )
*   Support YouTube/Vimeo videos. ( NEW )

**How to use**

1. First installed and active the plugin
2. Create a new Category
3. Create new slides
4. Include carousel in your in any content. e.g. post, page, widget, theme
5. Set Default Options from Plugin Settings Menu


= For use in post or page :- =
<code>[lgx-carousel]</code>

= For specific category please use category slug in shortcode as :- =
<code>[lgx-carousel cat="my-cat-slug" ]</code>

= For use template php file:- =
<code><?php echo do_shortcode('[lgx-carousel]'); ?></code>


**You can also use custom TinyMCE button to insert shortcode.**

== Installation ==


### Uploading in WordPress Dashboard

1. First download the plugin ZIP file from wordpress.org
2. Log in to your website administrator panel
3. Go to the 'Add New' in the plugins dashboard, click “Upload Plugin”
4. Upload [WordPress Slider](http://wordpress.org/plugins/lgx-owl-carousel) ZIP file by choosing it from your download location
5. Click Install Now button
6. Then click Activate the Plugin button.
7. You can see the plugin installed on Wordpress left menu.

### Using The WordPress Plugin Dashboard

1. Go to the 'Add New' in the plugins dashboard
2. Search for 'lgx-owl-carousel' or 'OWL Carousel Slider'
3. Click Install Now button
4. Then click Activate Plugin button
5. You can see the plugin installed on Wordpress left menu.

Now you can configure your Plugins options from settings.
